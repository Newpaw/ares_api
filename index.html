About the API

The My API is designed to provide data on Czech companies using the ARES system. It offers information about company details, including identification number (ICO), name, address, postal code, legal form, business fields, and size.

To access the API, you can use the provided endpoints and parameters to retrieve the desired company information.

Sample Code

Here's an example of how you can use the API to retrieve company data:

import requests

url = "https://ares.novopacky.com/company/{company_id}"
response = requests.get(url)

if response.status_code == 200:
    data = response.json()
    # Process the retrieved data
    # ...
